UNREAL ENGINE

Unreal Engine Version 4.27

This year I took the challenge of learning how to use Unreal Engine 4, a game development tool to create video games. Since is used by many studios all around the world, I decided
to learn how to use the platform to make my own games.

FIRST PROJECT:

I started by creating my first level. I created the room by making an empty box and positioning the player in the middle of the room. Then, I changed the texture of the floor, walls, and ceiling. Afterward, I wanted to make it cozier so I added some furniture. For this, I had to move around different views to position the furniture in the right place (making sure it was not in the air nor inside the floor). For an additional touch, I added a statue (which I changed the material to gold) and a picture to the wall (which I had to import, position, and scale).

SECOND PROJECT:

For the second project, I designed a more elaborated level. This level has stairs, a second floor, windows, doors, lights, as well as a space for a future elevator.

First, I started by creating a ground, big enough for a building. Then I created 3 hollow boxes, one for the main room, and one for each side (East and West).
Second, I placed the stairs in the desired position and leading to the second floor. Next, I created the second floor in each room by placing a box and changing the scale. I adjusted the stairs and floor to match and later made the space for the elevator.
Third, It was time to make the cutouts for the doors and windows. For this, I used the box brush but with the subtractive attribute.
Fourth, I changed the texture of the interior and exterior walls and floor, as well as the ceilings and the elevator cutout.
Fifth, Time to decorate! I added a door frame, a glass window (to be the door), and a handle (which I attached to said door). This was made this way so afterward they can be sliding doors. For the interior doors, I used a regular door with a door frame. Then, I placed the windows with their frames.
Finally, adding the lights. I added point lights for all the interior lightning and light sources for the exterior light (all pointing to the building).

GETTING FAMILIAR WITH BLUEPRINT:

For this phase, I am getting familiar with the visual programming used in UE. Thus, getting familiar with the blueprint editor.
Here I learned how to make different functions, translating java code into this visual format.
Event graph is often used as the main method, that calls other functions. Therefore, you make other functions and then connect them on the event graph, where event BeginPlay is.
I learned there are all the available types (int, booleans, float, etc) and different math functions like addition, subtraction, etc. You can have returns, as well as print statements. In this phase I went around playing with different methods I could create. I used if statements, loops, and nested loops.

THIRD PROJECT:

For this project I return to the sophisticated level design I did on phase 2 and added functionality plus landscape.
As I mentioned on phase 2, the main doors are going to be sliding doors, for me to do this I found that a trigger box was the option. When the character steps into this box the doors open, and when the character steps out they close automatically. Hence, I placed a trigger box bigger and wider than the main entrance doors, and opened the blueprint editor.
First, I started by adding a timeline to control for how long the door should move to open. Thus, I added a timeline with a vector track (to keep track of the object's movement, in this case the doors) for each door.
Second, I established the doors location. For this vector variables were used, as well as, getActorLocation action. This, to defined variables and stored the location of all four glass doors.
Third, it was time to make the trigger boxes work (when character enters trigger box do something, in this case open the doors). So, the values from the vectors (on the timeline) were used to add/subtract from the location of one of the doors to update its location along the timelineâ€™s timing. Now the doors work!
Finally, for the landscaping I used the landscape mode to create mountains and valleys. In addition, I used "Water Planes" (a water mesh), and "Landscape Pro 2.0 Auto-Generated Material" for the trees and bushes.
